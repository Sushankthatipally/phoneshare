# iPhone Web Shell

The Safari/PWA iPhone surface has been retired and removed from the active workspace.

## Status

- There is no longer an active `apps/iphone-web` build target in this repository.
- The native mobile scaffold in `apps/mobile` replaces the browser-first iPhone path.
- Any historical notes about the old web shell should be treated as archival only.

## Migration note

The old browser flow depended on manual pairing, camera APIs, PWA install prompts, and a local web backend. None of those pieces belong in the native-only plan.
